# MinigamesCore
A core created for the minigames server.

## Plugin usage:
Permission|Command|Description
---|---|---
core.broadcast|/bc <message>| Broadcast message in server
core.chatstaff|/s <message>| Send message in chat staff
core.clear|/clear <player>| Clear a player inventory
core.enderchest|/enderchest <player>| View an enderchest
core.fly|/fly| Turn on or off your fly mode
core.gamemode|/gamemode <gamemode>| Change your gamemode
core.god|/god| Make you invencible from other players
core.vanish|/vanish| Hide you from other players
core.invsee|/invsee <player>| view an inventory
core.jail|/jail <player>| Jail a player
core.jail|/unjail <player>| Drop a player who is stuck
core.say|/say <message>| Broadcast message with your name 
core.setspawn|/spawn set| Set the server spawn
core.speed|/speed <speed>| Change your speed
core.teleport|/tp <player>| Teleport you to a player
core.teleport|/tphere <player>| Teleport player to you
core.teleport|/tploc <world> <x> <y> <z>| Teleport into world
core.bypassjoin| ... | Not broadcast when player enter
